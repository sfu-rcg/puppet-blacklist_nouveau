# puppet-blacklist_nouveau

## Overview 

Blacklist the open source nVidia graphics driver (nouveau).


### Requirements

* A supported operating system:

  * CentOS 6 (tested)
  * Ubuntu >= 15.x (tested)

## Usage

```
include blacklist_nouveau
```
